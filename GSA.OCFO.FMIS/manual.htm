FMIS Reference


  • The FMIS User Guide provides users with comprehensive descriptions and step-by-step instructions in the following areas: Application Access and Log-in, Valid Accounting Transaction (VAT) data collection process, Queries, Most Frequently Used Queries, and Saving and Printing Data. The FMIS User Guide also provides users with a variety of Quick Tips that provide instructional and background information on important concepts and trouble-shooting.

  • The FMIS Training Presentation contains PowerPoint slides that are presented in the classroom course. These slides provide users with the content required to understand the data, VAT data collection process, and update frequencies. Users will also learn how to access FMIS and be able to distinguish between the different log-in screens, and understand how to compare and contrast the various queries available for data gathering and ways to manipulate query data. Finally, users will be able to navigate through network drives in order to save and print the reports that are generated in FMIS.

  • Quick Reference Guides provide users with instructions on specific FMIS features and topics listed below.
