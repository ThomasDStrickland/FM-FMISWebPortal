What's Changed for FMIS?

Please review this document to see what changes came to FMIS as part of the recent Pegasys 7.1.2 Upgrade.

NEW CITRIX LOGIN SCREEN

FMIS to launch new Citrix login portal the evening of December 21

FMIS users will see a new Citrix login screen after GSA IT completes a front-end upgrade the evening of Monday, December 21.

Once the upgrade is complete, the previous Citrix portal URL (https://xawi.gsa.gov) will automatically redirect to the new portal, which will be located at https://finhrapps.anywhere.gsa.gov. Users will use the same login information to log in to the system. Only the appearance of the login screen and the URL of the portal will change.

AGENCY CODE CHANGES IN FMIS DUE TO BAAR PHASE 2A IMPLEMENTATION

The Financial Management Information System (FMIS) now identifies an agency with a five-digit code. This transition to use a five-digit code to identify agencies is a part of the Government-wide Treasury Account Symbol Adjusted Balance System (GTAS) project and was part of the recent implementation of the Phase 2A Billings and Accounts Receivable (BAAR) module.

To find an agency’s new identification code, review the document titled "4-Digit to 5-Digit Agency Code Crosswalk (FMIS)" at https://insite.gsa.gov/portal/content/513050.

What’s available for FMIS users?

  • A new computer-based training course for FMIS is available. Learn how to use this important financial query tool

  • The Office of Financial Management Systems has updated their FMIS learning materials and resources available for all FMIS users. These resources include a User Guide, a Training Presentation, and Quick Reference Guides. The User Guide provides a comprehensive description on specific FMIS topics along with step-by-step instructions. The User Guide also provides a variety of Quick Tips that offer instructional and background information on important concepts and answers to help troubleshoot common issues. The Training Presentation contains PowerPoint slides that are presented in the classroom course. These slides provide specific instructions and descriptions of when and how to use FMIS features. The Quick Reference Guides provide users with quick instructions on specific FMIS features and topics. These topics include Frequently Used Queries, Queries, Saving and Printing, System Access and Log-In, and VAT Data Collection.

    To access the FMIS Reference Materials copy and paste the following URL in an Internet Explorer Window. http://cfo.fmis.gsa.gov/manual.htm (FMIS is not accessible using the Google Chrome browser application.)
