FMIS Citrix Plugin Download
     
   
Internet Explorer
If you are using Internet Explorer, the plug-in should load automatically and you can now return to the previous page and run your application.

If the installation does not start, please click here to download the file, click "Open from Current Location", and install the plugin manually.

NOTE : This download may take a few minutes, especially if done over a modem. Please watch the status bar at the bottom of your browser screen and allow the download to complete before leaving this page.

Once the plugin installation has completed, click here to be taken to the FMIS application.
Netscape Navigator
If you are using Netscape Navigator, follow these steps :

1. Click here to download the web client file.

2. When prompted by your browser, click on "Open from current location".

3. Allow the installation to complete.

4. Close and restart your browser.

5. Return to this page and click here to continue to the FMIS application.
