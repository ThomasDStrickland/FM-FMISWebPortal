Password Change for Compliance with Security Audit

To comply with GSA IT Security Policy (CIO P 2100.1C) requirements within the FMIS database and application; each database user will be obliged to change their password in the application every 90-days. This is the second password once you have logged on to the Citrix MetaFrame.

This activity can be accomplished by using the change password function within the FMIS application. At the menu, click on File and choose, Change Password....





Performing this change every 90-days will be necessary to retain access to the FMIS database and application. If you experience any difficulty making this change, please contact OCFo Service Desk via email OCFOServiceDesk@gsa.gov or phone 1-866-450-6588.

We apologize in advance for any inconvenience that this may cause.
