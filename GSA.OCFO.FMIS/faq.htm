Password Help
  • The password must be at least 8 characters in length and include letters, numbers and a special character.
  • Passwords are case sensitive, usernames are not.
    • Example password: Par1234#
  • Acceptable special characters are:` ~ ! @ # $ % ^ & * ( ) _ + - = { } | [ ] \ : " ; ' < > ? , . /
  • You will be notified if your password has been changed successfully.
